快捷键速查

掌握基础快捷键能显著提升效率：

  • 锁屏：Control + Command + Q
  • 定位URL：Command + L
  • 新建窗口：Command + N
  • 删除文件：Command + 回车
  • 最小化/关闭窗口：Command + M/W
  • 退出程序：Command + Q
  • 终端双击复制：选中文本后双击自动复制
  • 终端新建Tab：Command + T
  • 同应用多窗口切换： Command + ~
  • 同应用窗口列表：ctl + 朝下箭头
